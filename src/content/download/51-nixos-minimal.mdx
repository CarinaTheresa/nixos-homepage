You can install NixOS on physical hardware by burning one of the CD images onto a blank CD/DVD disk, or [by copying it onto a USB stick](/manual/nixos/unstable#sec-booting-from-usb). For **installation instructions**, please see the [manual](/manual/nixos/unstable).

### Minimal ISO image

The minimal installation ISO image **does not contain the graphical user interface,** and is therefore a lot smaller. You have to [run the installer from the console](/manual/nixos/unstable/#ch-installation). It contains a number of rescue tools.
